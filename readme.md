# Dog Breed Classifier

## Original Data Set

[Standard Dogs Dataset](http://vision.stanford.edu/aditya86/ImageNetDogs/)

### Summary:

* 120 dog breeds
* ~150 images per class
* Total images: 20,580

## Modified Data Set

### Summary

* 10 dog breeds
    * `afghan-hound`
    * `bernese-mountain-dog`
    * `great-pyrenees`
    * `irish-wolfhound`
    * `leon-berg`
    * `maltese-dog`
    * `pomeranian`
    * `samoyed`
    * `scottish-deerhound`
    * `shih-tzu`
* ~210 images per class
